## üöÄ Quick Start Guides

### For Codex (Autonomous Implementation)
1. Read [PROGRESS.md](./PROGRESS.md) to see what's next
2. Review the implementation guide for that PR
3. Check [BACKLOG.md](./01-active/BACKLOG.md) for acceptance criteria
4. Follow [WORKFLOW.md](./WORKFLOW.md) for prompts
5. Implement, test, commit, and update docs

### For Claude (Code Review & Planning)
1. Review PRs created by Codex
2. Provide critical feedback (see [WORKFLOW.md](./WORKFLOW.md))
3. Plan complex features requiring architectural decisions
4. Update documentation after review

### For Humans (Coordination)
1. Check [PROGRESS.md](./PROGRESS.md) for overall status
2. Use templates in [WORKFLOW.md](./WORKFLOW.md) to run sessions
3. Review and merge PRs
4. Update [PROGRESS.md](./PROGRESS.md) after each PR
